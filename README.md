# 🎬 Sentiment Analysis Web App

This project is a **Streamlit web application** that predicts the sentiment of movie reviews using Natural Language Processing (NLP).

Users can enter any movie review, and the app will classify it as **positive 😊** or **negative 😠**.

---

## 🚀 Live Demo

👉 https://sentiment-analysis-web-apps.streamlit.app

---

## 📌 Features

• Real-time sentiment prediction  
• Simple and interactive UI  
• Built using machine learning pipeline  
• Fast and accurate predictions  

---

## 🧠 Machine Learning Model

The model is built using:

• TF-IDF Vectorization  
• Logistic Regression  

Pipeline:

```text
Text → Cleaning → TF-IDF → Logistic Regression → Prediction
